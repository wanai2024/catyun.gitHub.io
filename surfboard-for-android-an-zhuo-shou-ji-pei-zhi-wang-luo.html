Surfboard For Android 安卓手机配置网络

Surfboard (冲浪板) For Android 安卓手机配置网络

下载软件

Surfboard为开源软件

  • 在 NextNet网盘 或 Github 里找到最新版本下载，可根据您设备处理器架构来下载 Surfboard-mobile-universal-*.apk。

方法一： 获取订阅地址并自动配置（四步走）

安卓手机浏览器输入 go.NextNet.one （图A0），选择 订阅 ➔ 我的订阅 ，找到“一键订阅”按扭，点击『导入到 Surfboard』,自动弹出 Surfboard app ，点击 NextNet 配置（图A1），返回，连接运行（图A2）。点击『代理』选择自已合理的节点（图A3），就像这样就完成了。

方法二：手动配置软件

由于安卓系统兼容性问题，可能会不能唤起APP，这样不能完成自动配置，出现这样的情况我们需要手动来配置。

  • 首先，安卓手机浏览器输入 go.NextNet.one，选择 订阅 ➔ 我的订阅 ，找到“一键订阅”按扭，点击『复制订阅地址』。
